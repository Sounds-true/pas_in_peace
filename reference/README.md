## Почему reference папка?

**Проблема:** Большие документы (100KB+) забирают весь контекст AI ассистентов и затрудняют работу.

**Решение:** Переместить их в reference/ и использовать только при необходимости.

**Результат:**
- ✅ Быстрая работа с актуальными планами
- ✅ Меньше путаницы
- ✅ Сохранен полный context для deep dive
